## Introduction

In this section you could see a design system made for my application ,
this system is apply to react components and html component across my
admin panel app and my portfolio page.

## Typography

### Scale

Typografies have 5 differents size , the size was created from a phibonacci scale on 16px base

<div className='flex flex-col gap-3 py-8' style={{backgroundColor:'var(--primary--500)'}}>
  <Typography size={TypographySize.bodySmall}>Body small 9.8px</Typography>
  <Typography size={TypographySize.bodyLarge}>Body large 16px</Typography>
  <Typography size={TypographySize.titleSmall}>Title small 25.8px</Typography>
  <Typography size={TypographySize.titleMedium}>Title medium 41.8px</Typography>
  <Typography size={TypographySize.titleLarge}>Title Large 68.2px</Typography>
</div>

### Font family

All text, from titles to paragraphs have **Noto sans font**, this is a sans serif font.
